What this website is all about?



It is basically a ticketing tool, which has mainly three roles

  1. Customers
  2. Engineer
  3. Administrator

Customers

  • Create account.
  • Email verification before activation of account.
  • Raise a request
  • Check status of submitted request
  • Request Call back(Submit Phone Number and our engineers would call you back).
  • Submit feedback for closed request.

Engineers

  • Provide solution to those tickets.
  • Search Frequently asked questions.
  • Add/Update Frequently asked questions.
  • Callback Customers.

Administrator

  • Create Engineer Account.
  • Assign tickets manually.
  • Take action in case of negative feedback.
  • View report.
  • Take action if request is not closed within a day.
